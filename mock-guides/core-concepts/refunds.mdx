# Refunds

[Placeholder for refunds documentation]

## What are Refunds?

Refunds allow you to return money to your customers for completed transactions.

## Refund Types

### Full Refunds
Return the complete transaction amount.

### Partial Refunds
Return only a portion of the transaction amount.

## Refund Process

1. **Initiate refund request**
2. **Process refund**
3. **Funds returned to customer**
4. **Refund confirmed**

## Refund Timeline

- Typical processing: 5-10 business days
- Depends on payment method
- Varies by card issuer
